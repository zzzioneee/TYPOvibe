# 바이브코딩 아이디어 파인더

## 트리거
사용자가 "아이디어 주라", "뭐 만들까", "주제 추천", "오늘 작업" 등을 말하면 이 로직을 실행한다.

## 로직

### 1. 날짜 계산
- 오늘 날짜 기준 d-2 ~ 오늘(d-0) 범위의 뉴스/칼럼/노래를 탐색
- 예: 오늘이 5/19이면 5/17~19 기준으로 탐색

### 2. 탐색 범위
다음 카테고리에서 바이브코딩으로 접목할 수 있는 주제를 탐색한다:
- 디자인 / 아트 / 패션 / 건축
- 기술 / AI / 사회현상
- 음악 (신보, 앤섬, 바이럴 노래)
- 흥미로운 세상사 / 요즘 이슈
- 시사하는 바가 있는 메시지 / 문화 트렌드

**제외**: 전쟁, 정치적 갈등, 민감한 역사적 사건 등 무거운 시사

### 3. 결과 포맷
5개 아이디어를 아래 형식으로 제공:

```
**① 제목**
- 소재: 어떤 기사/노래/이벤트인지 한 줄 요약
- 컨셉: 바이브코딩으로 어떻게 풀지 — 인터랙션, 모션, 타이포그래피 방식
- 링크: 실제 기사 URL (홈페이지 아닌 기사 직링크)
```

5개 중 1개를 추천하고 이유를 짧게 설명한다.

### 4. 링크 검증
- 홈페이지로 떨어지는 링크는 제공하지 말 것
- 기사 직링크가 있는지 확인 후 제공
- 페이월이 있으면 페이월 없는 대체 매체 링크 우선 제공

### 5. 작업 맥락 참조
- 이 프로젝트(TYPOvibe)는 매일 하나의 뉴스/이슈/노래를 바이브코딩으로 표현하는 작업
- 기존 작업 목록(WORKS 배열)을 참조해 중복되거나 너무 유사한 컨셉은 피한다
- 디자이너가 타이포그래피/인터랙션/모션으로 표현하기 좋은 주제를 우선 선정
