# Day 5 — No Pain (실리카겔)

**날짜**: 2026.04.22
**헤드라인**: 실리카겔 — No Pain
**출처**: [THE FIRST TAKE (YouTube Shorts)](https://www.youtube.com/watch?v=dFsBob8SLlg)

---

## 컨셉

실리카겔의 'No Pain'은 "시간의 흐름을 산다는 것. 질주하는 시간을 부숴버리려는 그들의 독립선언문"이라는 컨셉의 곡이다.

핵심 메시지: '현실을 살고 있는 사람들은 내가 만든 집 안에서 나만의 길을 외치고 있는 사람들을 봤을 때 무슨 생각을 할까? 실리카겔은 그 사람들을 대표해 말한다. 난 오늘 그 현실을 떠날 거야, 날 미워하지 마.' 약해져 있는 사람들끼리 유대관계를 쌓을 수 있게 내가 만든 집에서 노래합시다라는 의미.

이 곡의 핵심 가사 구간(20초)의 **실제 사운드웨이브 파형을 추출**하여, 각 알파벳의 획을 파형 shape로 표현. 종이 질감 위에 청춘들의 거친 독립선언문 스타일로 구현. 독립선언문은 원래 낭독하는 것인데, 타이포가 파형 그 자체로 표현되었으니 자연스럽게 낭독하는 것처럼 느껴진다.

---

## 구현 내용

### 1. 사운드웨이브 파형 타이포그래피 (Canvas)
- 실제 MP3에서 파형 데이터 추출 (Python wave 모듈 → JSON)
- 각 영문 알파벳의 획(stroke)을 수동 정의 (N: 3획, o: 1획 등)
- 획을 따라 파형 진폭 데이터를 매핑 — 사운드가 큰 부분은 획이 두껍고 들쭉날쭉, 조용한 부분은 가늘어짐
- 각 가사 라인의 실제 타이밍 구간 파형을 사용 (No pain = 0~1초 파형, Nowhere = 6~7초 파형 등)

### 2. 독립선언문 레이아웃
- 종이 텍스처 배경 (paper.png) 위에 프러시안 블루(#013DFD) 텍스트
- 영문 가사(파형 타이포): 크고 과감한 배치, 자유로운 줄바꿈
- 한글 가사(Pretendard Bold + 블러): 영문 사이 빈 공간에 빽빽하게 채움
- 전체 가사를 반복 사용하여 독립선언문의 빼곡한 느낌 표현
- 검은 배경 + 중앙 세로 종이 (3:4 비율) — 선언문 문서 느낌

### 3. 인터랙션: 음악 재생 + 파형 순차 전환
- 클릭하면 음원 재생 시작 (20초, 18초부터 fade out)
- 재생 전: 모든 영문이 균일 두께 직선 (파형 없는 깔끔한 기하학적 형태)
- 가사 타이밍에 맞춰 **글자 단위로 순차 전환**: N→o→p→a→i→n 순서로 직선→파형
- 전환 완료된 글자: 최종 파형 모습으로 고정 + 실시간 앰프 진동 (사운드에 맞춰 두께 미세 변화)
- 노래 끝: 모든 글자가 파형 상태로 고정, 진동 멈춤

### 4. 가사 타이밍
| 시간 | 가사 |
|------|------|
| 0~1초 | No pain, no fail |
| 1~2초 | 음악없는 세상 |
| 5.2~6.2초 | Nowhere |
| 7~8초 | no fear |
| 8~10초 | 바다 같은 색깔 |
| 11~12초 | No cap |
| 12~13초 | no cry |
| 14~17초 | 이미 죽은 사람 아냐 |
| 17~19초 | 사실 |

### 5. 음원 처리
- YouTube Shorts에서 30초 음원 추출
- ffmpeg로 20초 컷 + 18~20초 fade out 적용
- 저작권: 비상업 포트폴리오, 20초 이내, 출처 명시

## 기술 스택

| 기술 | 용도 |
|------|------|
| Canvas 2D | 파형 스트로크 드로잉, 텍스트 렌더링 |

## 삽질 포인트

1. **파형 스타일 탐색**: 파형을 획에 매핑하는 방식을 여러 번 바꿈. 수직 방향 fill shape → 수평 빗살 → 블롭 → 고해상도 → 저해상도. 결국 저해상도(초당 200포인트) + 50포인트 샘플링 + 수직 fill shape가 가장 자연스러웠음.

2. **한글 레이아웃 배치**: 영문 파형의 실제 렌더링 크기를 코드에서 정확히 알 수 없어서, 한글 텍스트 좌표를 수차례 조정. 최종적으로 포토샵에서 직접 배치한 좌표를 코드에 반영하는 방식으로 해결.

3. **글자별 순차 전환 타이밍**: "No"와 "pain,"이 같은 start 시간이라 동시에 활성화되는 문제. 단어를 별도 LYRICS 항목으로 쪼개고 start/end를 세분화해서 해결.

4. **파형 구간과 활성화 타이밍 분리**: waveStart/waveEnd(파형 데이터 구간)와 start/end(글자 활성화 타이밍)를 별도로 관리해야 했음.

5. **한글 폰트 호환성**: 박경리체(OTF), 김훈체(TTF) 등 여러 폰트를 시도했으나 브라우저 파싱 에러. woff2 변환으로 해결.

6. **마무리 효과 실패**: 독립선언문을 낭독 후 종이를 구겨버리는 효과(스탑모션 PNG 4장)와 불태우는 효과(Canvas 그라디언트)를 시도했으나, 사각형 종이→구겨진 종이 전환이 부자연스럽고 Canvas 불 효과도 어색해서 제거.

---

## 시도했지만 빠진 것들

- **마우스 스크러빙 낭독**: 마우스 Y 위치로 음원 시간을 제어하는 인터랙션. 음원이 끊겨서 어색해 제거.
- **마우스 자석 효과**: 커서 주변 파형 포인트가 끌려가는 효과. 컨셉과 연결성이 없어 제거.
- **한글 파형 스트로크**: 한글도 영문처럼 파형으로 표현하려 했으나, 획 분해가 복잡하고 픽셀 외곽선 추출 방식은 가로 줄무늬가 됨. Pretendard Bold + 블러로 대체.
- **종이 구겨지기/불태우기**: 노래 끝 마무리 효과. 구겨진 종이 PNG 스탑모션과 Canvas 불 효과 모두 시도했으나 부자연스러워서 제거.

## 동료 공유용 요약

**왜 만들었나**: 실리카겔 'No Pain'의 "질주하는 시간을 부숴버리려는 독립선언문" 컨셉을 타이포그래피로 표현. 실제 사운드웨이브 파형을 알파벳 획에 매핑하여, 소리가 글자의 형태가 되는 경험.

**핵심 인터랙션**:
- 클릭하면 음악 재생 + 가사 타이밍에 맞춰 글자가 직선→파형으로 순차 전환
- 전환된 글자는 사운드에 맞춰 미세하게 둥둥 진동 (앰프 느낌)
- 종이 텍스처 위에 영문 파형 타이포 + 한글 가사가 빽빽하게 — 독립선언문

**기술**: Canvas 2D + Web Audio API (AnalyserNode) + Python 파형 추출

**삽질**: 파형 스타일을 7번 이상 바꿈 (수직/수평/블롭/빗살/고해상도/저해상도). 한글 레이아웃은 포토샵에서 직접 배치한 좌표를 코드에 반영. 종이 구겨지기/불태우기 효과는 시도했지만 부자연스러워서 제거.
