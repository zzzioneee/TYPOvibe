# Day 2 — 로봇개가 개를 산책시켜..

**날짜**: 2026.04.17
**헤드라인**: "로봇개가 개를 산책시켜.."
**출처**: [조선일보](https://n.news.naver.com/mnews/article/023/0003970949)

---

## 컨셉

"로봇개가 개를 산책시켜"라는 헤드라인에서 **로봇 ↔ 개**의 대비감을 타이포그래피로 표현.
- **텍스트** = 로봇 (딱딱하고 기계적, 삐걱거리는 자모 분리)
- **강아지** = 진짜 개 (러프한 핸드드로잉, 폴짝폴짝 살아있는 모션)
- 둘을 **목줄**로 연결해서 산책하는 장면을 모션 타이포그래피로 구현.

---

## 구현 내용

### 1. 강아지 (Canvas SVG 스타일)
- 몸통/머리/귀/다리/꼬리를 Canvas에 직접 그림
- 면처리(fill) + 러프한 선(jitter)으로 핸드드로잉 느낌
- 다리가 폴짝폴짝 움직이고 꼬리 흔들림
- 마우스 올리면 직접 컨트롤 가능, 기본은 자율 움직임
- 강아지 주변에 원형 마스크 → 공원 배경이 보임 (radialGradient 블러)

### 2. 텍스트 (한글 자모 분리 + GSAP)
- "로봇개가 / 개를 / 산책시켜···" 3줄 중앙 정렬
- 한글 유니코드 분해로 초성/중성/종성 분리
- 글자별 다양한 모드:
  - **split**: 자모가 독립적으로 삐걱 회전 (steps ease, 기계적)
  - **whole**: 통째로 미세 기울어짐 (sine ease, 부드럽게)
  - **static**: 안 움직임
  - **split_wide**: 자모 분리 + 간격 넓게 (ㄱ과 ㅐ 겹침 방지)
  - **glitch**: RGB 분리 지지직 효과
- Pretendard Black (900) 사용
- 가운데점(···) 3개 — 각각 다른 크기, 미세한 높낮이

### 3. 목줄
- Canvas quadraticCurveTo로 곡선
- 강아지 목 → 텍스트 첫 글자(ㄹ) 연결
- 거리에 따라 처짐(sag) 변화

### 4. 인터랙션
- 텍스트가 우→좌로 일정 속도 스크롤 (무한 루프)
- 강아지를 마우스로 끌면 텍스트 속도 변화
- 강아지가 오른쪽으로 버티면 텍스트 전체가 좌우로 미세하게 부들거림 (textShake)
- 좌우 페이드아웃으로 자연스러운 진입/퇴장

### 5. 배경
- 전체 블랙 배경
- 강아지 주변만 원형 마스크로 공원 배경(하늘+잔디+나무) 노출
- 마스크 경계에서 핑크/퍼플/시안/골드 별 모양 파티클 반짝임
- VHS 주사선 오버레이

---

## 기술 스택

| 기술 | 용도 |
|------|------|
| GSAP 3 + EasePack | 자모 삐걱거림, steps/rough ease |
| Canvas 2D | 강아지 드로잉, 배경, 마스킹, 파티클 |
| 한글 유니코드 분해 | 초성/중성/종성 분리 (0xAC00 기반) |
| Pretendard Black | 텍스트 폰트 |

---

## 삽질 포인트

1. **한글 자모 분리**: 유니코드 0xAC00~0xD7A3 범위에서 초성(19)/중성(21)/종성(28) 인덱스 계산. 종성이 없는 글자(가, 개 등)는 2개만 반환.

2. **Canvas 텍스트 중앙 정렬**: 각 줄 너비를 미리 계산해서 가장 긴 줄 기준으로 offsetX를 줘야 함. 점(·)과 일반 글자의 blockW가 다르면 계산이 틀어짐.

3. **강아지 ↔ 텍스트 물리**: 처음엔 텍스트를 강아지 위치에 직접 연동했더니 새총처럼 튕김. 텍스트를 독립 스크롤로 분리하고, 강아지 속도만 텍스트 속도에 영향 주는 방식으로 해결.

4. **마스킹**: destination-out으로 블랙 오버레이에 구멍 뚫기. radialGradient로 블러 경계.

5. **루프 전환**: 텍스트가 왼쪽 밖으로 나가면 오른쪽에서 다시 등장할 때, 강아지 위치도 같이 리셋해야 스프링 튕김이 안 생김.

---

## 동료 공유용 요약

**왜 만들었나**: "로봇개가 개를 산책시켜"라는 헤드라인의 로봇/개 대비감을 타이포그래피로.

**핵심 인터랙션**: 
- 한글 자모가 분리되서 삐걱거리는 로봇 텍스트
- 핸드드로잉 강아지가 목줄로 텍스트를 끌고 감
- 마우스로 강아지를 컨트롤하면 텍스트 속도/떨림 변화

**기술**: GSAP steps ease + Canvas 2D + 한글 유니코드 분해

**삽질**: 텍스트-강아지 물리 연동이 가장 어려웠음. 직접 연동하면 새총, 독립 스크롤+속도 영향으로 해결.
