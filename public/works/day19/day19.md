# Day 19 — 똑똑, 울산에 자리 있나요?

**날짜**: 2026.05.20  
**태그**: Canvas, Leaflet, Data, Interactive

## 개요

2025년 겨울 울산 태화강·동천·회야강을 방문한 철새가 전년 대비 36.5% 증가한 111종 121,733마리를 기록했다.  
시베리아·몽골·중국·일본·호주 등 세계 각지에서 출발한 철새들이 울산 태화강으로 모여드는 이동경로를 인터랙티브하게 시각화한 작품.

## 기술 스택

- **Leaflet.js + CartoDB Dark** — 다크 테마 세계지도 위에 Canvas 오버레이
- **Canvas 2D** — 이동경로 점선 + 철새 SVG 실루엣 렌더링
- **CatmullRom 보간** — 자연스러운 곡선 이동경로 생성
- **SVG 새 실루엣 5종** — `source-in` composite으로 대륙별 네온 컬러 적용

## 구현 포인트

- 대륙별 네온 컬러 그룹핑: 러시아(시안), 몽골(퍼플), 중국(옐로우), 일본(핑크), 캄차카(그린), 호주(오렌지)
- 경로 그라디언트: 출발지 어둡게 → 울산 도착할수록 밝아지는 HSL 그라디언트
- Flock 파라미터: 경로당 N마리 철새가 일정 간격으로 줄지어 이동
- 줌 레벨 연동: 줌 아웃 시 선 굵기·새 크기가 자동으로 작아짐
- 경로 캐시: 지도 이동/줌 이벤트 시 재계산, 중간엔 캐시 유지

## 컨트롤

| 파라미터 | 설명 |
|---|---|
| Count | 이동경로(트랙) 수 |
| Speed | 철새 이동 속도 |
| Bird Size | 새 크기 배율 |
| Flock | 경로당 철새 수 |
