# Day 6 — 절약도 공감도 함께

**날짜**: 2026.04.23
**헤드라인**: 절약도 공감도 함께
**출처**: [오픈애즈 — 절약도 공감도 함께](https://www.openads.co.kr/content/contentDetail?contsId=19285)

---

## 컨셉

고물가 시대, 소비자들은 정보와 감정을 '혼자' 소비하지 않는다. 온라인 담타·음식만안와요·거지맵 세 서비스가 3월을 기점으로 동시 확산 중이며, 소비자가 직접 만든 '초경량 공감 커뮤니티'가 빠르게 퍼지고 있다. 절약과 감정 해소를 함께 나누는 새로운 소비자 연대 방식의 부상.

"절약도 공감도 함께" 헤드라인을 픽셀 폰트(Galmuri9)로 크게 표현한 뒤, 글자를 구성하는 블록들이 트랜스포머처럼 분해·재조립되면서 스마일 얼굴들로 변태. 사람들이 모여서 와글와글 밍글링하고 강강수월래를 하는 모습으로 — "함께"라는 키워드를 시각적으로 체현.

---

## 구현 내용

### 1. 픽셀 텍스트 → 스마일 얼굴 모핑 (Canvas)
- Galmuri9 픽셀 폰트로 "절약도 / 공감도 / 함 께" 헤드라인 렌더링
- 각 글자를 BLK(7px) 단위 블록으로 분해하여 좌표 추출
- 스마일 얼굴도 동일한 BLK 블록으로 정의 (외곽 원 + 눈 + 입)
- 텍스트 블록 좌표 → 얼굴 블록 좌표로 1:1 매핑, cubic easing으로 보간
- 글자별로 매칭하여 크로스 이동 최소화 (절→절 근처 얼굴, 약→약 근처 얼굴)

### 2. 동전 얼굴 (4종 표정)
- 외곽: 울퉁불퉁한 원 (시드 기반 랜덤 변형, 0.96~1.04 범위)
- 눈: 2블록씩 양쪽
- 입: 스마일 U자 / 일자 / O입 / 활짝 웃음 — 4종 랜덤 배정
- 크기: 75~110px 랜덤

### 3. 애니메이션 타임라인
| 시간 | 상태 |
|------|------|
| 0~1.5초 | 헤드라인 텍스트 정적 표시 |
| 1.5~2.5초 | 둠칙둠칫 들썩거림 (sin 진동, 점점 강해짐) |
| 2.5~4초 | 블록 모핑: 텍스트 → 스마일 얼굴 (cubic ease) |
| 4초~ | 밍글링: 자유이동 + 강강수월래 반복 |

### 4. 밍글링 모션
- 12초 주기: 자유이동(4초) → 강강수월래(4초) → 자유이동(4초)
- 강강수월래: 모든 얼굴이 하나의 원(반지름 160px) 위에서 기차놀이처럼 회전
- 자유이동: 랜덤 방향 + 벽 반사 + 충돌 회피 (얼굴끼리 겹치지 않게)

### 5. Orphan 블록 처리
- 텍스트 블록 수 > 얼굴 블록 수일 때, 남는 블록은 표정에 배분하지 않음
- 헤드라인 상태에서는 모든 블록 표시 (텍스트 안 잘림)
- 모핑 진행 시 남는 블록은 fade out으로 자연스럽게 삭제

### 6. 레이아웃
- 검은 배경 + 중앙 연두색(#cbff94) 캔버스 (3:4 비율)
- "함 께"는 양끝맞춤 (함: 좌측, 께: 우측)
- FROM 링크: 캔버스 상단 중앙, 출처 연결

## Kiro 삽질 포인트

1. **텍스트 잘림 문제**: 얼굴 수를 줄이면 텍스트 블록이 남아서 "함께"의 일부가 사라짐. 남는 블록을 얼굴에 추가 배분하면 표정이 깨짐. 최종 해결: 남는 블록을 모핑 시 fade out으로 삭제하되, 헤드라인 상태에서는 전부 표시.

2. **표정 깨짐**: 남는 텍스트 블록을 랜덤 얼굴에 배분하면 눈/입 위치에 블록이 추가돼서 표정이 이상해짐. 배분 자체를 제거하고 orphan fade out 방식으로 전환.

3. **크로스 이동**: 전체 블록을 정렬 매칭하면 "절" 블록이 "께" 근처 얼굴로 날아가는 문제. 글자별로 매칭하여 해결.

4. **얼굴 하단 잘림**: 캔버스 높이 늘리기, 강강수월래 반지름 줄이기 등 시도했으나 근본 원인은 블록 매칭 문제였음. orphan 처리로 해결.

---

## 시도했지만 빠진 것들

- **자소 분리 → 사람 형상화**: 한글 자소(초성/중성/종성)가 분리되면서 동전 얼굴로 변태하는 컨셉. 자소 형태와 얼굴 파츠가 안 맞아서 블록 모핑 방식으로 대체.
- **얼굴 내부 채우기**: 블록 수를 늘리기 위해 원 내부를 채웠으나, 내부 블록에 텍스트가 흡수되면서 외곽/표정이 사라짐.

## 동료 공유용 요약

**왜 만들었나**: 고물가 시대 소비자 연대 트렌드 기사에서 "절약도 공감도 함께"라는 헤드라인을 픽셀 타이포로 표현. 글자가 분해되어 동전 사람들로 변태하고, 함께 어울리는 모습으로 "함께"를 시각화.

**핵심 인터랙션**:
- 픽셀 헤드라인이 둠칙둠칫 들썩이다가 블록 단위로 분해
- 블록들이 재조립되면서 동전 얼굴 16명으로 변태
- 얼굴들이 화면을 돌아다니며 밍글링 + 강강수월래

**기술**: Canvas 2D + Galmuri9 픽셀 폰트 + 블록 모핑 애니메이션

**Kiro 삽질**: 텍스트 블록↔얼굴 블록 수 불일치로 텍스트 잘림/표정 깨짐 반복. orphan fade out 방식으로 해결.
