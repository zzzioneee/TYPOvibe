## 출발

뉴닉에서 '영희페스티벌' 기사를 읽고, 여성 창작자들의 '영광과 기쁨(Glory and Joy)'을 시각적으로 표현하고 싶었음. 단순한 축제 포스터가 아니라, 레코드판이 고속 회전하며 색이 번지는 듯한 역동적인 에너지를 만들어보자는 아이디어.

## 컨셉

**"Glory and Joy"** — 여성 창작자들이 삶과 무대에서 누려야 할 영광과 기쁨. 화려한 꽃 그래픽들이 레코드판처럼 여러 중심축으로 회전하며, 색이 원형 궤적을 따라 smear되는 인터랙티브 타이포그래피.

레퍼런스: 러시아 댄스 페스티벌 포스터 — 여러 중심점의 radial spin blur가 화면 전체를 지배하며, 형태가 고속 회전 잔상으로 길게 늘어지는 시각 효과.

## 구조

- 흰 배경 + 40개의 다양한 꽃 형태(블랍/페탈/잎/별꽃)가 화면 가득
- 5개의 독립적인 vortex(회전축)가 각각 자체 offscreen canvas에서 렌더
- 각 vortex: 소스 이미지를 45개 미세 각도로 회전 복사 → 원형 clip 마스크로 합성
- "Glory and Joy" 텍스트: 소스에도 포함(일부 회전 영향) + 최종 위에 선명하게 오버레이
- 마우스 인터랙션: 가까운 vortex가 가속 회전
- 노이즈 그레인 텍스쳐 오버레이 (CSS SVG filter)

## 기술

### Multi-Layer Spin Smear (Canvas 2D)

핵심 기법: 각 vortex를 **독립적으로** 렌더하고 합성.

1. **소스 이미지** (srcCanvas): 40개 꽃 + 텍스트를 1920×1080에 한 번 그림
2. **vortex별 offscreen canvas** (5개): 각각 소스를 45번 미세하게 다른 각도로 회전 복사 (drawImage + translate/rotate) → 균일 opacity
3. **원형 clip 마스크**: 각 vortex의 영향 범위를 `ctx.arc() + ctx.clip()`으로 제한
4. **메인 canvas에 합성**: 5개 vortex layer를 순서대로 drawImage

```
소스(꽃+텍스트) → vortex1(45 rotated copies, clip) → vortex2 → ... → vortex5 → 텍스트 오버레이 → 노이즈
```

### 꽃 형태 (4종)

- **drawBlob**: bezier curve 기반 유기적 비정형 형태
- **drawPetal**: 방사형 타원 꽃잎 (4~8개)
- **drawStarFlower**: 가느다란 뾰족 꽃잎 방사
- **drawLeaf**: bezier 티어드롭/잎 형태

색상 팔레트: 핫핑크/마젠타 + 네온 연두/라임 + 퍼플/라벤더 + 에메랄드

### 마우스 인터랙션

```javascript
const boost = 1 + Math.max(0, 1 - mouseDist / v.r) * 3;
// 마우스가 vortex 중심에 가까울수록 최대 4배 회전 가속
```

### 노이즈 텍스쳐

CSS로 SVG feTurbulence 노이즈를 overlay blend로 6% opacity 적용. 필름 그레인 질감.

### 스케일링

1920×1080 고정 캔버스 + `Math.max(vw/W, vh/H)` cover 방식 → 여백 없이 뷰포트 채움.

## 삽질 기록

### v1: Three.js 파티클 모핑 (폐기)
처음엔 "Glory" → "and" → "Joy" 텍스트를 파티클로 모핑하는 Three.js 작업을 시도. 완성했지만 네가 원하는 방향(레코드판 spin blur)과 달라서 v2로 전환.

### v2-1: WebGL 셰이더 spin blur (다수 실패)
- 처음 8샘플 → 블러 안 보임
- 30샘플 → 뭉개짐 (gaussian blur처럼)
- 60샘플 → GPU 크래시
- distance-weighted vs closest-center → voronoi 경계선 발생
- 2-pass framebuffer → 여전히 "부드럽게 뭉개짐"
- 근본 문제: 셰이더의 spin blur는 "형태가 늘어지는" 게 아니라 "색이 평균되는" 것

### v2-2: Canvas 2D 다중 회전 복사 (채택)
- drawImage + rotate를 45번 반복 → 형태의 회전 잔상이 남음
- 5개 독립 vortex + clip 마스킹 → 개별 레코드판 느낌
- 소스에 꽃 형태가 있어야 잔상이 유기적으로 보임
- 균일 opacity (gaussian weight 제거) → "비늘" 대신 연속 호

### 텍스트 처리
- 소스에만 넣으면 완전히 회전되어 안 읽힘
- 소스에서 빼면 회전과 무관하게 뜨서 어색
- **이중 배치**(소스 + 오버레이)가 정답 — 일부는 회전되고 일부는 선명

## 핵심 교훈

### WebGL spin blur ≠ 형태 smear
셰이더의 spin blur는 각 픽셀을 원형 궤적 이웃과 **평균**내는 것. 결과는 "부드러운 번짐". Canvas의 다중 회전 복사는 형태를 **그대로 반복**하는 것 → 형태의 윤곽이 살아있는 잔상. 레퍼런스가 원하는 건 후자.

### 소스 이미지의 색 대비가 핵심
어떤 blur/smear를 쓰든, 소스의 색 경계가 선명해야 결과물에서도 "띠"가 보임. 부드러운 그라데이션 소스에 blur 먹이면 무조건 탁해짐.

### 독립 레이어 렌더 + 합성이 단일 셰이더보다 유연
"화면 전체에 한 번에 효과 적용" vs "각 영역을 따로 처리하고 겹침" — 후자가 레이어별 독립적인 회전 방향/속도 표현에 훨씬 적합.

### 마우스 인터랙션은 간단해도 효과적
거리 기반 speed boost만으로도 "내가 이 레코드판을 돌리고 있다"는 직관적 피드백이 됨.

## 동료 공유용 요약

### 한 줄 요약
레코드판처럼 회전하는 여러 소용돌이 위에 꽃과 "Glory and Joy" 텍스트가 얹혀진 인터랙티브 타이포그래피.

### 왜 이걸 만들었나
영희페스티벌의 '영광과 기쁨'이라는 메시지를 레코드판이 돌아가며 색이 번지는 역동적인 비주얼로 표현하고 싶었음. 마우스를 가까이 대면 해당 레코드가 더 빨리 돌아감.

### 작업 흐름
1. 레퍼런스(러시아 포스터) 분석 → "다중 중심 spin smear" 효과 확인
2. Three.js 파티클 모핑 v1 시도 → 방향 전환
3. WebGL spin blur 셰이더 다수 시도 → "뭉개짐" 문제로 실패
4. Canvas 2D 다중 회전 복사 방식 채택 → 형태 잔상 유지
5. 5개 독립 vortex 렌더 + clip 합성
6. 텍스트 이중 배치 (회전 영향 일부 + 선명 오버레이)
7. 마우스 인터랙션 + 노이즈 텍스쳐

### 핵심 인터랙션
- **레코드판 회전**: 5개 vortex가 각각 다른 속도/방향으로 계속 회전
- **마우스 근접 가속**: 가까운 vortex가 최대 4배 빨리 돌아감
- **텍스트 이중 노출**: 일부는 회전에 묻히고 일부는 선명 → 읽히면서도 역동적
