# Day 20 — 절창

**날짜**: 2026.05.21  
**태그**: Canvas, Typography, Editorial, Animation

## 개요

구병모 소설 『절창』의 조판 구조를 드로잉으로 재해석한 실험적 편집 디자인 작업.

책의 활자 레이아웃에서 우연히 만들어지는 불규칙한 형태미 — 들여쓰기, 줄 바꿈, 문장 길이에 따른 ragged edge — 를 시각적 예술 요소로 끌어올렸다.

텍스트를 '읽는 정보'가 아닌, 지면 위에 배치된 **조형물**로 인식하게 만드는 타이포그래피 실험.

## 기술 스택

- **Canvas 2D** — 텍스트 렌더링 + 외곽선/구멍 드로잉
- **measureText** — 행 단위 실제 픽셀 너비 측정
- **requestAnimationFrame** — 순차 드로잉 애니메이션
- **Noto Serif KR** — 본문 서체

## 구현 포인트

- 단락 전체를 하나의 불규칙 외곽선으로 감싸기 — 들여쓰기로 파인 움푹한 형태, 짧은 줄의 ragged edge가 외곽선의 형태가 됨
- 문장부호(쉼표, 마침표 등) 위치에 불규칙 폴리곤 구멍 — Positive/Negative 공간의 경계를 노골적으로 드러냄
- 진입 시 라인 드로잉 애니메이션 — 왼쪽 페이지 → 오른쪽 페이지 순서로 펜이 그리듯 외곽선 등장
- 종이 질감 배경 이미지로 실제 책 페이지 느낌 구현
- 두 페이지 레이아웃 — 좌(p.20) / 우(p.21) 책 펼침 형태

## 인터랙션

- 진입 시 자동 드로잉 애니메이션 시작
- 클릭 시 처음부터 재생
