# Day 4 — 불황 속의 웨이팅?

**날짜**: 2026.04.20
**헤드라인**: "불황 속의 웨이팅? 우리는 왜 줄 서는 것에 열광하는가"
**출처**: [이오플래닛](https://eopla.net/magazines/41553)

---

## 컨셉

지갑이 얇아진 저성장 시대, 현대인은 실패 없는 소비를 위해 기꺼이 시간을 지불하며, 줄 서서 기다리는 행위 자체가 새로운 럭셔리가 되었다는 뉴스에서 영감.
헤드라인 텍스트로 인간의 형태를 만들고, 팀 단위로 줄 서서 웨이팅하다 입장하는 모션 타이포그래피.

---

## 구현 내용

### 1. 텍스트 인간 조형물 (Canvas)
- 머리/몸통/팔/다리 각 파츠에 헤드라인 글자가 채워짐
- 파츠별 다른 글자 크기로 조형성 표현 (머리: 큰/중/2줄, 몸통: 중간, 팔다리: 작은)
- 두꺼운 외곽 스트로크로 shape가 두드러짐
- 머리 3종 랜덤: 1글자 큰 / 2글자 몸통크기 / 1글자×2줄
- 글자 채우기 순서: 머리→몸통→다리→팔 (헤드라인이 자연스럽게 읽히도록)

### 2. 기차 시스템 (줄 서기)
- 모든 팀이 하나의 행렬로 왼쪽 밖에서 오른쪽으로 이동
- 팀 단위 입장: A팀(3명) [간격] B팀(2명) [간격] C팀(3명)...
- 맨 앞 팀이 오른쪽 끝에 도착하면 멈춤 (웨이팅)
- 뒤에 2팀 이상 쌓이면 맨 앞 팀 퇴장 → 나머지 앞으로 이동 → 새 팀 추가
- 우측정렬: 먼저 온 팀이 항상 오른쪽(앞)

### 3. 걷기 모션
- 팔다리가 어깨/엉덩이 기준으로 진자 회전 (옆에서 본 걷기)
- 2종 걷기 모드 랜덤: 교차 보행(X자) / 같은 쪽 동시(// \\)
- 사람마다 걷기 빈도, 진폭, 바운스, 속도 전부 랜덤
- 대기 시: 팔다리 스윙 없이 상하 바운스만 (lively한 느낌)

### 4. 팀별 컬러
- 원색 + 파스텔/뉴트럴 믹스 10색
- 번갈아 배치 (원색→파스텔→원색→파스텔)로 톤 겹침 방지
- 매 루프마다 셔플되어 다른 조합

### 5. 헤드라인 배분
- "불황 속의 웨이팅?" / "우리는 왜 줄 서는 것에 열광하는가" 2종
- 팀 내에서 번갈아 배정 + 셔플

## 삽질 포인트

1. **줄 서기 순서**: 각 사람의 걷기 속도가 달라서 도착 순서가 뒤섞힘. 개별 이동 대신 "기차" 방식(전체가 같은 속도로 이동)으로 해결.

2. **팀 간 겹침**: 여러 팀을 동시에 스폰하면 위치 계산이 꼬여서 겹침. 한 행렬로 미리 배치하고 전체를 이동시키는 방식으로 해결.

3. **ㅇ 안쪽 채우기**: Canvas strokeText가 닫힌 곡선(ㅇ, ㅎ) 안쪽까지 침범. fillText를 3번 그려서 해결.

4. **컬러 톤 겹침**: 랜덤 색상이 비슷한 톤으로 연달아 나옴. 원색/파스텔을 번갈아 배치하는 방식으로 해결.

## 동료 공유용 요약

**왜 만들었나**: "웨이팅 문화"를 다룬 뉴스에서 영감. 줄 서서 기다리는 행위를 텍스트 인간 조형물로 시각화.

**핵심 인터랙션**:
- 헤드라인 텍스트로 만든 인간들이 팀 단위로 줄 서서 웨이팅
- 팀별 다른 컬러 외곽 스트로크
- 걷기 모션 2종 (교차/같은쪽) + 사람마다 다른 속도/진폭

**기술**: Canvas 2D + 기차 시스템 (행렬 이동)

**삽질**: 줄 서기 순서가 가장 어려웠음. 개별 이동→동시 도착→기차 방식으로 3번 재작업.
