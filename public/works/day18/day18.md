# Day 18 — 햄찌의 분노

**날짜**: 2026.05.19  
**태그**: Canvas, Interactive, Game

## 개요

AI툴로 직장인 회사생활 애니메이션을 만드는 유튜버 '정서불안 김햄찌'에서 영감받은 인터랙티브 바이브코딩.  
2000년대 초중반 추억의 고전게임 'Desktop Destroyer (바탕화면 부수기)'를 햄찌 테마로 재해석했다.

햄스터 커서로 맥북 화면을 발톱 공격, 불지르기, 파멸펀치로 박살낸다.

## 기술 스택

- **Canvas 2D** — 파괴 이펙트 렌더링
- **ImageData 픽셀 조작** — 타격 지점 각진 구멍 생성
- **불꽃 파티클 시스템** — 딜레이 큐로 드래그 경로 따라 시간차 피어오름
- **그을음 누적** — 연속 선 + 바깥 퍼짐으로 자연스러운 번짐

## 툴

| 툴 | 단축키 | 동작 |
|---|---|---|
| 발톱공격 | Q | 클릭 → 3~4줄 대각 긁힘 자국 |
| 불지르기 | W | 드래그 → 그을음 + 시간차 불꽃 |
| 파멸펀치 | E | 클릭 → 각진 구멍 + 균열 |
