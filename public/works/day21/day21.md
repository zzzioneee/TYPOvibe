# Day 21 — 낯선 사람들이 모이면

## 날짜
2026.05.22

## 컨셉
'커뮤니티'를 정해진 계획대로 굴리는 제품이 아니라, 사람 간의 관계 속에서 끊임없이 변화하고 성장하는 역동적인 유기체로 바라보는 시선을 시각 작업으로 표현.

## 메타포
연결 · 연대 · 연속 — 줄기, 뿌리, 덩굴, 손, 꽃, 사과, 애벌레, 나무 등 자연물

## 텍스트
낯선 사람들이 / 모이면 / 어느 순간 / 서로가 답이 돼요

## 기술
- React (in-browser Babel)
- SVG 조형 활자 (16개 자모 글리프)
- SVG filter: feTurbulence + feDisplacementMap (지글지글 손떨림)
- SVG filter: graffiti-noise (노이즈 텍스쳐)
- 한글 초중종성 레이아웃 시스템 (V/H/VB/HB)
- 타이핑 애니메이션 (리듬 랜덤화: 빠른 버스트 + 느린 구간)
