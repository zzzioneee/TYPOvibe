# Day 25 v2: 우아한판 — 게임 시나리오

## 기본 정보

| 항목 | 내용 |
|------|------|
| 세션 시간 | 10~15분 |
| 동시 접속 | ~3,000명 |
| 디바이스 | Android / iOS 혼합 |
| 팀 구성 | 6팀 (레드-페퍼로니 / 옐로우-치즈 / 브라운-불고기 / 오렌지-파프리카 / 블랙-올리브 / 그린-바질) |
| 출력 채널 | 무대 LED 모니터 + 개인 모바일 |
| 진행 | MC가 무대에서 모니터/모바일 보며 멘트로 분위기 조성 |

## 서사 구조

1막(팀 빌딩) → 2막(화면 연출) → 3막(War Mode)

> 피자 토핑 = 각 팀. 토핑으로서 팀 빌딩 → 토핑이 모여 한 판의 민트 피자 → 원팀 정신으로 War Mode 돌입 → 우리의 판으로 물들이기

### 1막: 팀 빌딩 (재료 분류 게임)

| 시점 | HOST (무대 LED) | SERVER | USER (모바일) |
|------|----------------|--------|---------------|
| 1-1 | MC: 게임 시작 안내 | 전체 "시작" 신호 브로드캐스트 | 시작 신호 수신 → 게임 시작 |
| 1-2 | 타이머 표시 (30초 카운트다운) | 서버 기준 타이머 운영, 개인 결과 수신 | 30초 타임어택: 상단에서 팀 재료 낙하, 신선/상한 분류 (버리기/킵하기 버튼). 좌상단 타이머, 우상단 게이지 |
| 1-3 | 타이머 10초 이하: 흔들림 강조 연출 | - | 타이머 10초 이하: 흔들림 강조 연출, 난이도 상승 |
| 1-4 | 타이머 종료 | 팀별 결과 집계 (팀원 중 일정 비율 이상 성공 시 팀 성공) | 화면 딤드 → 재료 등장 + 개인 성공/실패 표시 |
| 1-5 | MC: "성공 여부를 다함께 확인해 볼까요?" → 6팀 결과 노출 | 팀별 성공/실패 판정 결과 전송 | 개인 성공/실패 화면 노출 (무대 주목) |

**성공 판정 기준:** 팀원 N명 중 일정 비율(예: 88%) 이상 개인 성공 시 팀 성공 (버퍼 비율 TBD)

**미확정 사항:**
- 개인 성공 기준 (게이지 몇 % 이상?)
- 팀 성공 버퍼 비율 (88%? 80%?)
- 실패 시 어떻게 되는지 (재도전? 감점? 연출만?)

### 3막: War Mode (미니게임 연속)

| 시점 | HOST (무대 LED) | SERVER | USER (모바일) |
|------|----------------|--------|---------------|
| 3-1 | 민트 피자 → 3천여 개로 증식 → 서울 지도 형태로 배치 → 갑자기 빨간색 전환 (위기 연출) | 3막 시작 준비 | 변신 중이던 민트 피자가 빨간색으로 전환 (위기 연출) |
| 3-2 | MC: "우리 다시 힘을 모아야겠죠?" 게임 안내 | 미니게임 큐 준비 | 대기 → 게임 준비 화면 |
| 3-3 | 미니게임 N 진행 표시 + 참여 팀 안내 | 미니게임 시작 신호 + 결과 수신 | 미니게임 플레이 (해당 팀) / 응원 화면 (비해당 팀) |
| 3-4 | 성공 시: 빨간 피자 → 민트 피자로 색 전환 (점진적) | 성공 판정 → 상태 업데이트 | 성공/실패 피드백 |
| 3-5 | (3-3 ~ 3-4 반복, 3~4회) | - | - |
| 3-6 | 100% 민트 달성 → 승리 연출 | 최종 상태 브로드캐스트 | 모바일 화면도 민트색 전환 → 승리 화면 |

**미니게임 진행 방식:** 사전 순서 자동 진행 (HOST 실시간 선택 아님)

**민트 전환 비율:** 미니게임 1개 성공 = 고정 % (총 N개 미니게임 확정 시 100/N으로 계산)

**미니게임 구성 (3~4개, 내용 TBD):**

**미확정 사항:**
- 미니게임 구체적 내용

## 백엔드/프론트 뼈대 관점 필요 기능

### SERVER
- WebSocket 기반 실시간 통신 (3천 동접)
- 전체 시작/종료 신호 브로드캐스트
- 서버 기준 타이머 동기화
- 팀별 결과 집계 + 성공 판정 로직
- 미니게임 큐 관리 (어떤 게임, 어떤 팀, 시작/종료)
- 무대 모니터용 상태 API (점유율, 타이머, 팀 결과)

### FRONT (모바일)
- QR 진입 → 팀 선택 → 대기 화면
- 서버 신호에 따른 화면 전환 (1막/2막/3막)
- 1막: 재료 분류 게임 UI + 결과 전송
- 2막: 영상/GIF 재생
- 3막: 미니게임 컴포넌트 (플레이 / 응원 분기)
- 승리 화면

### FRONT (무대 LED)
- 별도 URL (관리자/호스트 뷰)
- 서버 상태 실시간 반영 (타이머, 팀 결과, 점유율)
- 연출 영상 재생 트리거
- MC용 진행 컨트롤 (선택)
